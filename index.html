O QUE É O BOX DA MODA ?

O box da moda é um clube onde todo mês você recebe itens incríveis em sua casa para unboxing. Você pode escolher o box que mais lhe agradar, tendo cada um, itens diferenciados.

Ficou interessado não é? Então role para baixo e veja o que você irá receber...

O QUE VEM NO BOX DA MODA ?

Cada box vem com conteúdo diferente, sendo entre eles os seguintes:

Uma peça de ROUPA!

As peças de roupa são de qualidade, sendo que em cada pacote pode variar a marca.

Um ACESSÓRIO!

Todo pacote acompanha um acessório para dar aquele diferencial em você!

Um MIMO!

Para aqueles que curtem uma surpresinha temos os mimos que iram lhe surpreender.
